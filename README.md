# HIRage

A terminal Rust compiler explorer built with Ratatui.

HIRage lets you inspect Rust code through multiple compiler stages directly from your terminal:

- Source
- HIR
- MIR
- LLVM IR
- Assembly

## Features

- File browser
- Function browser
- HIR viewer
- MIR viewer
- LLVM IR viewer
- Assembly viewer
- Keyboard navigation
